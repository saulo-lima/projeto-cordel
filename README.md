# projeto-cordel
# 📜 Projeto Cordel Moderno

Este projeto foi desenvolvido como parte do **Desafio 12 do Curso em Vídeo**, com o objetivo de recriar um cordel em formato de página web utilizando HTML e CSS.

##  Acesse o projeto

🔗 https://saulo-lima.github.io/projeto-cordel/

---

## 🧠 Objetivo

Praticar e consolidar conceitos fundamentais de desenvolvimento front-end, transformando um conteúdo textual em uma experiência visual moderna e responsiva.

---

## 🛠️ Tecnologias utilizadas

* HTML5 (estrutura semântica)
* CSS3 (estilização)
* Google Fonts (fontes externas)

---

## 🎨 Técnicas aplicadas

* ✨ Efeito **Parallax**
* 🔤 Uso de **fontes externas**
* 🎯 **Variáveis CSS** (`:root`)
* 📱 Responsividade com `vw` e `clamp()`
* 🎭 Tipografia com `small-caps`
